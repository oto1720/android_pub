### DigestViewModel.kt

#### 変更の意図
API呼び出しを `viewModelScope.launch {}` で非同期実行する前に、同期的に `isLoading = true` をセットしてUIに即座に反映する。

#### 指摘事項

**[🟢 G-2] isLoading セットが同期的でコルーチン開始前に確実に反映される**

```kotlin
// ✅ launch {} の外で同期セット → UIがスピナーを即座に表示できる
fun showQuestion() {
    val current = _uiState.value as? DigestUiState.ReadMode ?: return
    _uiState.value = current.copy(isLoading = true)  // 同期
    viewModelScope.launch {                           // 非同期
        val question = generateQuestionUseCase(articleId)...
    }
}
```

> `launch {}` 内でセットすると最初の `runCurrent()` まで反映されないが、外に出すことでテストでも `runCurrent()` なしに確認できる。
